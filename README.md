# AI Voice Translator (Russian → English)

Синхронный голосовой переводчик с русского на английский язык с использованием Deepgram (STT), Gemini/Cherry API (LLM) и локального контейнера Kokoro (TTS).

---

## 1. Запуск контейнера озвучки (Kokoro TTS)

Для синтеза речи используется Docker-контейнер проекта **Kokoro FastAPI** (версия для CPU).

Запустите контейнер следующей командой:

```bash
docker run -d \
  -p 8880:8880 \
  --name kokoro-tts \
  --restart unless-stopped \
  ghcr.io/remsky/kokoro-fastapi-cpu:v0.2.2
```

После этого API озвучки будет доступно локально по адресу: `http://localhost:8880/v1`.

> [!NOTE]
> Так как контейнер использует только ресурсы процессора (CPU), синтез речи (TTS) для одной фразы может занимать от 2 до 4 секунд в зависимости от мощности вашего процессора.

Создайте файл `.env` в корневом каталоге проекта и укажите в нём конфигурацию:

```env
# API ключ Deepgram для распознавания речи
DEEPGRAM_API_KEY=14d1106a34a1bb4e52e75865d33c81b1b5e6d7ac

# Настройки перевода (Cherry API / Gemini)
LLM_URL=https://cherryapi.duckdns.org:1337/gemini-antigravity/v1
LLM_API_KEY=sk-8a19e1425c017ea808114f85f5e53f3c
LLM_MODEL=gemini-3.5-flash-minimal

# Настройки локальной озвучки (Kokoro FastAPI)
KOKORO_URL=http://localhost:8880/v1
```

---

## 3. Запуск веб-приложения

1. Установите зависимости проекта:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте приложение в браузере по адресу:
   ```
   http://localhost:3000
   ```

---

## 4. Особенности работы

* **Потоковая передача**: Переводчик автоматически нарезает речь на смысловые фразы по знакам препинания (запятые, точки, двоеточия) и отправляет их на обработку.
* **Дебаунс пауз**: Если вы делаете паузу в речи более чем на 1.0 секунду, накопившийся текст автоматически отправляется на перевод.
* **Направление аудио**:
  * Во вкладке **Input** выберите источник звука (ваш микрофон или виртуальный кабель для захвата звука из приложений).
  * Во вкладке **Output** выберите наушники/динамики для прослушивания перевода или «Виртуальный микрофон» для трансляции перевода в другие программы (например, Discord).
